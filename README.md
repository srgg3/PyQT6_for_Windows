📚 Программа для представления экзаменационного материала

Проект разработан для автоматизации процесса проверки и анализа результатов экзаменов с использованием Python и PyQt6.

🚀 Описание проекта

Основная идея проекта — создать удобное приложение, которое:
	•	автоматизирует проверку экзаменационных работ
	•	снижает вероятность ошибок при проверке
	•	упрощает работу экспертов
	•	предоставляет понятный и простой интерфейс

Приложение рассчитано на любого пользователя, имеющего компьютер.

⸻

🔐 Авторизация

В приложении реализована система входа:
	•	окно авторизации
	•	ввод логина и пароля
	•	проверка данных пользователя

⸻

🖥️ Интерфейс

Программа имеет простой и интуитивно понятный интерфейс:
	•	удобная навигация
	•	визуализация данных
	•	несколько окон для работы с результатами

⸻

🛠️ Технологии

Проект реализован с использованием:
	•	Python
	•	PyQt6
	•	SQLite

⸻

🧱 Архитектура

Основные компоненты:
	•	классы для работы с интерфейсом
	•	модули для обработки данных
	•	логика авторизации

⸻

🗄️ База данных

В проекте используется база данных SQLite со следующими таблицами:

👤 user
	•	id — идентификатор
	•	login — логин
	•	password — пароль

📊 result
	•	имя пользователя
	•	вариант
	•	результат (1/0 за каждое задание)

📝 answers
	•	ответы на все задания

⸻

📈 Возможности
	•	вход и регистрация пользователей
	•	хранение результатов экзаменов
	•	анализ ответов
	•	удобный просмотр данных

⸻

🔮 Планы по развитию

В дальнейшем планируется:
	•	добавление выбора предмета
	•	сравнение результатов с критериями оценки
	•	перевод первичных баллов во вторичные
	•	расширение функционала анализа

⸻

📌 Заключение

Использование Python и PyQt6 позволяет создавать эффективные и удобные приложения для автоматизации проверки экзаменов. Проект имеет потенциал для дальнейшего развития и расширения.
